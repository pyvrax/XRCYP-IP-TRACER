# 💻 About

XCRYP-IP is a professional IP lookup and geolocation tool written in Python.

It provides detailed public IP information directly from the terminal with a clean and stylish interface.

---

# ⚡ Features

* 🌍 IP Geolocation
* 🛰 ISP Detection
* 🏢 Organization Lookup
* 📡 ASN Information
* 🔍 Proxy/VPN Detection
* ☁ Hosting Detection
* 🗺 Google Maps Coordinates
* 💙 Stylish Blue CLI Interface
* 🖥 Windows & Linux Support
* ⚙ Fast API Requests

---

# 📦 Installation

## 🪟 Windows

Run:

```bash
requirements.bat
```

Start tool:

```bash
start.bat
```

---

## 🐧 Linux

Install requirements:

```bash
pip3 install -r requirements.txt
```

Run:

```bash
python3 main.py
```
